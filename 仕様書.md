## 設計ドキュメント

### users テーブル

ユーザー（従業員）情報を管理するテーブル

| カラム名   | データ型 | 制約                        | 説明                     |
| ---------- | -------- | --------------------------- | ------------------------ |
| id         | integer  | PRIMARY KEY, AUTO_INCREMENT | ユーザー ID              |
| name       | varchar  | NOT NULL                    | ユーザー名               |
| email      | varchar  | NOT NULL, UNIQUE            | メールアドレス           |
| password   | varchar  | NOT NULL                    | パスワード（ハッシュ化） |
| created_at | datetime | NOT NULL                    | 作成日時                 |
| updated_at | datetime | NOT NULL                    | 更新日時                 |

### posts テーブル

勤怠記録（出勤・休憩・退勤）を管理するテーブル

| カラム名   | データ型 | 制約                        | 説明         |
| ---------- | -------- | --------------------------- | ------------ |
| id         | integer  | PRIMARY KEY, AUTO_INCREMENT | ID 　　　　  |
| startWork  | datetime | NULLABLE                    | 出勤時刻     |
| startBreak | datetime | NULLABLE                    | 休憩開始時刻 |
| endBreak   | datetime | NULLABLE                    | 休憩終了時刻 |
| endWork    | datetime | NULLABLE                    | 退勤時刻     |
| created_at | datetime | NOT NULL                    | 作成日時     |
| updated_at | datetime | NOT NULL                    | 更新日時     |

## 開発環境構築手順

### 前提条件

以下をインストール：

-   PHP 8.0 以上
-   Node.js 18.0 以上
-   Composer、Git、SQLite3

**Linux でのインストール:**

```bash
sudo apt install -y php php-sqlite3 php-mbstring nodejs npm composer git
```

### セットアップ手順

```bash
# 1. リポジトリクローン
git clone https://github.com/xxxx/kintaikanri.git
cd kintaikanri

# 2. 環境設定
cp .env.example .env

# 3. PHP依存のインストール
composer install

# 4. アプリケーションキー生成
php artisan key:generate

# 5. データベース初期化
php artisan migrate:fresh --seed

# 6. Node.js依存のインストール
npm install

# 7. フロントエンドビルド
npm run build

# 8. 開発サーバ起動（ターミナル1）
php artisan serve --host=0.0.0.0 --port=8000

# 9. アセット監視（ターミナル2）
npm run dev
```

### アクセス

ブラウザで `http://localhost:8000` にアクセス

### トラブルシューティング

| 問題                 | 解決方法                                 |
| -------------------- | ---------------------------------------- |
| SQLite エラー        | `sudo apt install -y php-sqlite3`        |
| npm エラー           | `npm cache clean --force && npm install` |
| ポート 8000 使用中   | `php artisan serve --port=8001`          |
| データベースリセット | `php artisan migrate:fresh --seed`       |
